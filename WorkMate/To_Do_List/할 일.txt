O/X 퀴즈 : 
  게임 시작. 로딩화면에서 플레어들의 게임 준비 메시지를 받아 모든 유저가 체크가 되면 좌표배열로 모든 플레이어 위치 조정
  1. 로딩화면에서 플레이어들의 게임준비 메시지를 받아 모든 유저가 체크가 되면 게임을 시작(타이머 체크의 데이터를 보냄) is_breaking > is_breaking 코드를 실행함
  2. 서버에서 자체 타이머가 돈 후 출력될 문제와 is_during 메시지를 보냄 > 클라이언트에서 문제를 출력함
  3. 서버에서 자체 타이머가 돈 후 is_checking 메시지를 보냄 > 클라이언트에서 is_checking 실행함
  4. 문제가 끝날때까지 123번을 반복
  5. 모든 문제가 끝나면 서버에서 is_ending 메시지를 보냄 > 클라이언트에서 gameover 메시지를 보내고 점수를 정산중 화면을 띄워 대기시킴
  게임 오버. 서버에서 해당 방의 있는 모든 유저의 gameover 메시지를 받아 모두가 게임이 끝났다면 화면을 전환시킴
  확인 사항 (화면이 꺼졌을때 메시지가 어떤것을 받는지 {전에 받았어야할 메시지가 받아지는지 / 그다음 메시지가 받아지는지})

레이싱 게임 : 
  게임 시작. 로딩화면에서 플레어들의 게임 준비 메시지를 받아 모든 유저가 체크가 되면 클라이언트는 좌표배열로 모든 플레이어 위치 조정
  1. 서버는 보내야하는 초기값인 장애물의 시작 좌표와 방향 , 골인 지점 2개 좌표 , 아이템 좌표와 게임 시작 메시지( 타이머 )를 보냄 
      > 클라이언트는 초기값 데이터를 받아 화면에 출력시키며 타이머와 함께 시작
  2. 서버는 장애물이 없어지는 시간을 대충 재서 매 타임마다 새로운 장애물의 시작좌표와 방향을 보냄 ( 아이템도 동일 )
  3. 장애물과 아이템, 골인 지점은 모두 캐릭터 좌표와 겹칠때마다 메시지를 보내고 서버에서 처리한 뒤 뿌려줌
  4. 클라이언트에서 타이머가 다 되면 점수 정산중 화면과 함께 서버에 gameover메시지를 보냄
  게임 오버. 서버에서 해당 방의 있는 모든 유저의 gameover 메시지를 받아 모두가 게임이 끝났다면 화면을 전환시킴

뒤집기 : 
  게임 시작. 로딩화면에서 플레어들의 게임 준비 메시지를 받아 모든 유저가 체크가 되면 좌표배열로 모든 플레이어 위치 조정
  0. 화면 설계
  1. 모든 카드는 객체로 만들어서 배열에 저장
  1-1. 배열의 카드들을 캔버스에 그림
  2. 버튼을 누른 사람이 있다면 그 카드의 좌표를 모든 유저에게 보내서 뒤집어 보이게
  3. 총 맵의 크기는 10X5 50개의 배열 카드!!
  4. 각각의 유저들은 서버로부터 뒤집힌 카드들의 좌표와 유저들의 위치를 전송받음
  4-1 클라이언트에서 몇번째 카드를 뒤집었는지 (1번째인지 2번째인지) 1번째일때 0.5 2번째 1.5 서버에서 체크해주기 전까지 다른 카드를 뒤집을수 없게 작성
  5. 서버는 1번째인지 2번째인지 클라이언트에서 주는데 2번째이면 같은 종류인지 체크를해서 맞다 아니다 전달해서 점수 증가
  6. 시간제한은 180 초 ( 3분 )
  게임 오버. 서버에서 해당 방의 있는 모든 유저의 gameover 메시지를 받아 모두가 게임이 끝났다면 화면을 전환시킴
