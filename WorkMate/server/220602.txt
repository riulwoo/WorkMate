화면 전환 후 내부에서 js 로직을 체크하여 움직이는데 
내부 js 로직에서 서버와 주고받을 데이터들을 관리하는 로직을 서버에 분류하여 넣어놔야하고
그 데이터들을 이용해 서버에 있는 스코어 점수를 관리, 이후 게임이 종료되면 클라쪽에서 다시한번 화면 전환 요청을 보내고
서버에서 받아와 해당 화면을 전환 -> 이것을 3라운드까지 반복
이것을 반복하는데 각각의 게임마다 데이터들의 종류가 다르니 이것을 또 나눠서 각 게임마다의 데이터 주고받는 로직을 
만들어놔야함

서버에 넣어야할 코드들은 
1. 게임마다 데이터를 뿌려줄 코드 ( 총 3개 )
2. 라운드 종료 시 점수를 받아와 저장할 코드
3. 모든 라운드 종료 시 결과 창으로 전환함과 동시에 데이터를 보내는 코드

각 게임 클라이언트에 넣어야할 코드들은
1. 각 게임별 주고받는 데이터로직
2. 게임이 끝났을 때 데이터를 서버로 보내는 로직

라운드가 끝났을 때 html 하나
